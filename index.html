Operators

Operators

+, -, *, /, and % are called operators. An operator is a special character (or characters) that indicates an action to be performed.

Variables

Variables

var is short for variable. This is the traditional way of declaring a variable but it has some longstanding issues so we won't use this option. let is a more modern way to declare a variable whose value will change. const is a more modern way to declare a variable whose value will never change.

Variable Naming Conventions

Functions

Functions

A function is something that performs an action. Just like a method, a function can take an argument.

Methods

Methods

A method is an action run on a piece of data; you can think of it as a message you send to a piece of data, and the result is the response.

Arguments

Arguments

Arguments are passed into the parentheses (). It's not required for a method to have arguments. Some take no arguments, some take optional arguments, and some take one or even many arguments.

Parameters

Parameters

A parameter is a placeholder for an argument. We use parameters when we declare functions and methods. We'll learn more about declaring functions when we actually start creating our own custom functions soon.
